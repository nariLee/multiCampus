실전 예제 만들기

1.배열까지의 학습 내용을 review 하는 application
2. 구조
	1) pattern
		(1) MVC
		(2) DTO(VO, bean)
	2) 실전 개발 설계
		(1) main이 있는 시작 가능한 클래스
			 : customer.view.StartView.java
		(2) 데이터 표현용 클래스
			 : customer.domain.~DTO.java[~VO.java, ~Bean.java, ~.java]
		(3) 데이터 보유하고 삭제하고, 수정하는 등의 핵심 클래스
			 : customer.model.~Model.java
		(4) 어떤 요청인지 구분하는 컨트롤러 클래스
			 : customer.controller.~Controller.java
		(5) 결과값 출력하는 클래스
		     : customer.view.EndView.java

3. 실행 process
		StartView(파라미터O) -> Controller(파라미터O,void), -> Model(리턴O,파라미터O) -> Controller(파라미터O,void) -> EndView(파라미터O)
